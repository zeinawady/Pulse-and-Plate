 ```bash

 ```

 2. Install dependencies for both frontend and backend:

 ```bash
 cd client
npm install
cd ../server
npm install
```

3. Start the application:
 - In one terminal (for frontend):
 ```bash
 cd client
 npm start
 ```
 - In another terminal (for backend):
 ```bash
 cd server
 node server.js
 ```
